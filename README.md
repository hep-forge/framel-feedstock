[![Build & Upload](https://github.com/hep-forge/framel-feedstock/actions/workflows/autoupload.yml/badge.svg)](https://github.com/hep-forge/framel-feedstock/actions/workflows/autoupload.yml)

[![Anaconda Platforms](https://anaconda.org/hep-forge/framel/badges/platforms.svg)](https://anaconda.org/hep-forge/framel)

Feedstock for [framel](https://git.ligo.org/virgo/virgoapp/Fr) — part of [hep-forge](https://anaconda.org/hep-forge).
Builds linux-amd64 + linux-arm64 in one matrix workflow and uploads to the
[hep-forge](https://anaconda.org/hep-forge) Anaconda channel.

Frame library

## Install

```bash
conda install -c hep-forge -c conda-forge framel
```
